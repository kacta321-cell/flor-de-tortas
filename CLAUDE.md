# Flor de Tortas — Catálogo Web

## El proyecto
Catálogo web one-pager para Florencia, repostera de zona norte Buenos Aires.
Regalo de su hijo (el usuario).

## La marca
- **Nombre:** Flor de Tortas
- **Instagram:** @flor.d.tortas
- **Canal de venta:** WhatsApp
- **Zona de cobertura:** Pilar, Tortuguitas, Escobar, Nordelta (zona norte GBA)
- **Público:** Mamás y familias de nivel socioeconómico medio-alto de zona norte

## Productos
- Tortas de diseño con masa (fondant/decoradas)
- Torta Rogel
- Torta de números con flores
- Tortas frutables
- Bizcochuelos (cumpleaños, casamiento)
- Lemon pie
- Chocotorta
- Marquise
- Brownies
- Galletitas decoradas
- Mini postres

## Design system
- **Tipografía display:** Cormorant Garamond (serif elegante)
- **Tipografía cuerpo:** DM Sans
- **Paleta:**
  - Crema: #FAF6F0
  - Rosa polvillo: #E8C4B8
  - Salvia/verde suave: #B5C4B1
  - Dorado cálido: #C9A96E
  - Chocolate/texto oscuro: #3D2B1F
  - Blanco: #FFFFFF

## Stack
- Frontend: HTML/CSS/JS puro (sin framework)
- Hosting: Vercel (free tier) → flor-de-tortas.vercel.app
- No hay backend por ahora

## Archivos clave
- `index.html` — el one-pager completo
- `public/images/` — fotos de las tortas (a reemplazar con las reales)
- `vercel.json` — config mínima de deploy

## Para deployar a Vercel
```bash
npm i -g vercel
vercel --prod
```
